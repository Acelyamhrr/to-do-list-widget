# Widget Liste de Tâches

Un widget simple et léger pour gérer vos tâches quotidiennes et suivre vos progrès.

## Fonctionnalités

- Affichage de la date du jour
- Suivi de l'avancement avec une barre de progression
- Ajouter, cocher ou supprimer des tâches
- Interface responsive, adaptée à tous les écrans

## Fichiers du projet

- `index.html` : structure HTML principale  
- `styles.css` : styles et mise en page  
- `script.js` : logique et gestion des interactions  

## Installation

1. Télécharger ou cloner le projet :  
   ```bash
   git clone <URL_DU_PROJET>

2. Ouvrir index.html dans n'importe quel navigateur moderne.

## Utilisation

- Taper une tâche et cliquer sur Add Task ou appuyer sur Entrée.
- Cocher la case pour marquer la tâche comme terminée.
- Cliquer sur le X pour supprimer une tâche.
- La barre de progression se met à jour automatiquement en fonction des tâches complétées.
